No diretório raiz "cd "administracao-clientes"
abra o powershell e execute: docker-compose up --build

Dois contâiners serão criados:
- administracao.clientes.sqlserver
- administracao.clientes.api

Quando o container administracao.clientes.api for criado, uma migration automática será executada no container administracao.clientes.sqlserver.

Para acessar, basta abrir o SSMS e inserir os dados de acesso:

server name: localhost
user: sa
pass: !AsDqWe123@@
database name: AdministracaoClientes

Endpoints:

GET http://localhost/api/v1/clientes
POST http://localhost/api/v1/clientes
